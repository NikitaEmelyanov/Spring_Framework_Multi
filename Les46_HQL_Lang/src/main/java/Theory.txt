
_________________________________________________HQL___________________________________________________

Для некоторых задач приходится писать запросы:
1) Вывести людей, у которых возраст > 20
2) Удалить всех людей, у которых имя начинается на "T"
3) Обновить всех людей и всем поставить имя "Test"
4) ...

Для этого в Hibernate используется свой язык запросов - HQL

___________________________HQL vs SQL. Очень похожи, но есть различия_________________________________
                                              |
    Java Класс   Поле класса     HQL          |                 Таблица     Столбец        SQL
        |           |                         |                    |           |
FROM Person WHERE name = 'Test'               |   SELECT * FROM Person WHERE name = 'Test'
                                              |
В SQL мы работаем с таблицами, в HQL с сущностями (Java классы с аннотацией @Entity). В остальном
все очень похоже

1) HQL был разработан, чтобы мы не спускались до уровня таблиц, а делали запрос, используя наши
сущности (Java классы с аннотацией @Entity)
2) HQL конвертируется в SQL
3) В Hibernate можно использовать и обычный SQL для сложных запросов, но лучше HQL
4) Spring приложение - это обычно комбинация Hibernate (+Spring Data JPA) и SQL для нестандартных
запросов
